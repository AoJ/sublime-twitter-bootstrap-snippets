    Sublime Text Twitter Bootstrap Snippets

    A bunch of nicely described and categorized Twitter Bootstrap code snippets for Sublime Text editor.

    Installation

    1. Clone the repository into your packages folder.

    git clone git@github.com:devtellect/sublime-twitter-bootstrap-snippets.git

    2. Download the .zip file and unzip it into your ST2 packages directory.
